# Usage

```bash
curl -fsSL https://raw.githubusercontent.com/irammini/installer/refs/heads/main/installer.sh | bash -s -- <tool>
```
> Replace `<tool>` with your wanted tool

# Requirements

1. **Ubuntu/Debian Environment** (22.04+ is recommended)
2. **cURL**
3. **Bash**
4. **Sudo access** (For most installer, this is required)
